# AI Resume Generator

A React + Vite resume builder with a live CV preview and PDF download support.

## Features

- Dark blue app background with a centered white builder card
- Resume form fields for name, email ID, course, qualifications, short about, and extra activities
- Multi-select skills checklist
- Live clean CV-style resume preview
- PDF export using `jspdf` and `html2canvas`

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
